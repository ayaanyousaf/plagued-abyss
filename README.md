# Plagued Abyss
#### Author: Ayaan Yousaf
A 2D top-down wave-based survival shooter set in a dark abyss-themed dungeon level. A single player experience where the player must survive endless increasingly difficult waves of the infected, earn points, unlock new areas, and purchase upgrades to get stronger

▶️ [Demo Video](https://youtu.be/mvnx856TFW8?si=nuDfpZ704KnzGffW)

---

## ⚙️ How to Run
### Option 1 (Easiest)
1. Navigate to the ```/build``` directory
2. Open the folder for your OS (Windows or Linux)
3. Run the executable

### Option 2 (Run in Godot)
1. Clone the repository
2. Open the project in Godot 4.5.1
3. Click 'Run Project' or press 'F5'

---

## 🎮 Controls
- WASD - Move
- Mouse - Aim
- Left Click - Shoot
- F - Interact 
- ESC - Pause

---

## 🚩 Known Issues
- Enemies sometimes get stuck to player
- Bullets go through walls and have infinite range
- Bullets can be fired in any direction including behind the barrel
